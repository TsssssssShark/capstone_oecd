## Intructions to use the Interactive Map:
Download the entire `map/` folder and run `map.html`.

## .tex files:
`main.tex` is the main report and `summary.tex ` is the summary. Both should be rendered with XeTeX.

## `portals_information.csv`
Contains all data we collected

## Figures and tables:
`fig/` contains all the figures.
`tab/` contains all tables in .tex format.
`tab_png/` contains most tables in .png format.

## `data_report.qmd`:
Contains all codes to manipulate the data and produce the figures, tables, and the .js files for the map.
